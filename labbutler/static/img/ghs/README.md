# GHS hazard pictograms

Official UN GHS / EU CLP hazard pictograms, GHS01–GHS09.

Source: Wikimedia Commons (`GHS-pictogram-*.svg`), which mirrors the official UNECE
artwork. The pictograms are in the public domain (published by the United Nations;
ineligible for copyright as official government symbols).

| Code  | Meaning                     | Commons source file            |
|-------|-----------------------------|--------------------------------|
| GHS01 | Explosive                   | GHS-pictogram-explos.svg       |
| GHS02 | Flammable                   | GHS-pictogram-flamme.svg       |
| GHS03 | Oxidizing                   | GHS-pictogram-rondflam.svg     |
| GHS04 | Compressed gas              | GHS-pictogram-bottle.svg       |
| GHS05 | Corrosive                   | GHS-pictogram-acid.svg         |
| GHS06 | Acute toxicity (skull)      | GHS-pictogram-skull.svg        |
| GHS07 | Harmful/irritant (exclam.)  | GHS-pictogram-exclam.svg       |
| GHS08 | Serious health hazard       | GHS-pictogram-silhouette.svg   |
| GHS09 | Environmental hazard        | GHS-pictogram-pollu.svg        |

Used by the CAS→GHS lookup UI (`apps/inventory/ghs_lookup.py` maps PubChem pictogram
names to these codes) and intended for future label printing.
